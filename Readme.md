## 🧑‍💼 User Roles

### 👥 Regular Users
- Sign up / log in
- View and explore financial services
- Contact support
- Read blogs, FAQs, etc.

### 🛡️ Admin
- Access secure dashboard
- View, add, delete users
- Track stats and overview
- Add future modules (policies, blogs, etc.)
